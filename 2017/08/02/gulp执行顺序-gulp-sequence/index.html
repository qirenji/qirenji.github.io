介绍

Node.js 工具链中的典型任务运行器有 Grunt ，以及后起之秀 Gulp。

与以传统的方式进行配置的 Grunt 相比， Gulp 则使用“代码”的方式来描述任务。而它们另一个巨大的不同，则体现在运行方式上：Grunt 的运行方式很直观：解析依赖，使用配置逐步运行已定义的任务；而 Gulp 则默认将所有任务和步骤异步化运行。显而易见，Gulp 在效率上是有明显的提升的。
