解决冲突

Git最常用的可能就是多人协作时解决冲突问题。

一般会先push上自己的文件

然后发现推送失败了，这是因为远程的分支和自己的本地推送内容更新，所以需要使用git pull把远程分支与本地合并。

如果合并有冲突，则需要在本地解决冲突。

解决冲突之后，再重新git push就能推送成功了。

这就是多人协作之解决冲突。
