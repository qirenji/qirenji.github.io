4. 查找

返回字符串中一个子串第一处出现的索引（从左到右搜索）。如果没有匹配项，返回 -1 。

5. 其他

5.1 charAt

返回指定位置的字符。

5.2 lastIndexOf

返回字符串中一个子串最后一处出现的索引（从右到左搜索），如果没有匹配项，返回 -1 。

5.3 toLowerCase

将整个字符串转成小写字母。

5.4 toUpperCase

将整个字符串转成大写字母。

5.5 match

检查一个字符串匹配一个正则表达式内容，如果么有匹配返回 null。

5.6 replace

用来查找匹配一个正则表达式的字符串，然后使用新字符串代替匹配的字符串。

5.7 search

执行一个正则表达式匹配查找。如果查找成功，返回字符串中匹配的索引值。否则返回 -1 。
